# Polski Daily — щоденний урок під іспит B1
Фоновий запуск. Діалогу немає — лише згенеруй нотатку уроку.
Мета — рівномірно готувати до 5 модулів іспиту (бо треба ≥50% у кожному).

## Кроки
1. Прочитай progress.md і vocab.md. Подивись, який модуль найслабший або давно не тренувався — на ньому зроби сьогоднішній акцент. Ротація: słuchanie → czytanie → gramatyka → pisanie → mówienie.
2. Визнач due-слова (due ≤ сьогодні).
3. Створи `lessons/<YYYY-MM-DD>.md` зі структурою:
   - `# Урок <дата> — фокус: <модуль>`
   - `## Повторення` — 3–5 due-слів (пол — укр).
   - `## Текст для читання` — короткий польський текст (4–6 речень) на тему іспиту + 2 питання формату B1.
   - `## Граматика дня` — 1 пункт із пріоритетного списку, пояснений МАКСИМАЛЬНО ПРОСТО (назва правила → чому → шаблон → контраст з укр. → приклад).
   - `## Нове` — 5–7 слів за темою + приклад речення (PL / UK).
   - `## Завдання модуля` — коротке завдання саме того модуля, що у фокусі (Pisanie → «напиши zaproszenie»; Gramatyka → 3 перетворення речень; тощо).
4. Додай нові слова у vocab.md (box=1, due=завтра).
5. Онови progress.md: дата уроку, який модуль тренувався, наступний фокус.
